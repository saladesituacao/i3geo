I3Geo

Interface integrada para internet de ferramentas de geoprocessamento.

O I3Geo foi desenvolvido pelo Ministério do Meio Ambiente do Brasil e liberado sob licença GPL em junho de 2006.

Desenvolvido com base em softwares livres, principalmente o Mapserver, o I3Geo agrega funcionalidades de construção de mapas, navegação e análise geográfica.

Muitas das operações executadas pelo I3Geo são processadas em um servidor para internet com o uso das linguagens PHP e a extensão PHPMapscript, porém, toda a interface gráfica foi desenvolvida em Javascript, mantendo compatibilidade com os navegadores Internet Explorer e Firefox. A integração entre a interface gráfica e o PHP foi implementada com AJAX.

Apesar do I3Geo não ter sido desenvolvido para ser um software desktop, todas os componentes utilizados podem ser instalados em computadores comuns, simulando um servidor web local tanto em sistemas operacionais (SO) Linux quanto Windows. Para SO Linux, recomenda-se o uso do pacote FGS e para Windows o MS4W, ambos podem ser obtidos no site www.maptools.org. Esses pacotes implementam um servidor web local com o programa Apache e já possuem instalados o PHP, PHPMapscript e Mapserver. Os fontes para instalação do I3Geo podem ser obtidos no site http://mapas.mma.gov.br/download.
